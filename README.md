required to run
 1) mongo db  
 2) Any web broswer like chrome, firefox.
 3) Eclipse Ide or Intellj

Project Setup

 1) Import project to your workspace.
 2) Start Mongo db server
 3) Run Main class "StudentsMongodbApplication" which is inside the "com.student" package.
 4) Open "http://localhost:8080/".
 5) create two accounts one as admin and user and test it.
